Программа Simple KLADR предназначена для корректного заполнения адресов физических лиц при формировании налоговой отчетности или получения достоверного почтового адреса.
Программа может работать на любом компьютер с операционной средой Windows
Для работы программы нет необходимости устанавливать на компьютер базу данных, достаточно скачать готовый файлы  базы данных КЛАДР, например сайта ГНИВЦ (http://www.gnivc.ru/inf_provision/classifiers_reference/kladr/) и разместить их в любую папку на вашем диске.
Программа написана на языке C#,  компактна и не требует специальных знаний по установке.
Просто разместите файл delete_dir.exe в папку, которую необходимо очистить и нажмите кнопку "удалить".
Программа предназначена для свободного распространения.
